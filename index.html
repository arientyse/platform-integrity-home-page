GitAudit

It is an application that monitors the members of our GitHub organizations. Whether the member is allowed to stay or not is determined by some expandable policies:

  • It is required that you allow us to view your private, validated emails.
  • It is required that you must activate two factor authentication. (To learn more about 2FA, click here.)
  • It is required that your GitHub profile has your CareerBuilder email address.
  • It is required that you are an active employee.

Click here for more information and to check your compliance with these policies.

Datastore

Data store is a tracking tool used to track the activity of recruiters to help satisfy government audits. The tool collects data, which can used to generate a report for any given time frame and account.

The government program that runs this is called OFCCP. (Click here for more information about OFCCP.)

To generate a report, click here.
