Products

  • GitAudit

    It is an application that monitors the members of our GitHub organizations. Whether the member is allowed to stay or not is determined by some expandable policies: It is required that you allow us to view your private, validated emails. It is required that you must activate two factor authentication. It is required that your GitHub profile has your CareerBuilder email address. It is required that you are an active employee. Click here for more information and to check your compliance with these policies.

    Source Control

    We manage the source control systems for most of CareerBuilder.

    This is TFS for the matrix code (http://tfs.careerbuilder.com:8080/tfs/web) and GitHub for newer off-matrix code (https://github.com/cbdr).

    Team Portal

    The products are test environment, the automated test environment, test deployment, and production deployment.

    We are responsible for deploying code hourly to the test environment efficiently, for the health of the test environment, and for driving the tests developer write via our automated test environment that runs any valid unit and browser tests and reports the results to developers.

    You can view a summary and details of each test run on the team portal page.

    This page shows you at a glance:

    • which snapshot of code was deployed and tested successfully.
    • which set of code is the high water mark for what is currently in production.
    • which snapshot of code failed deployment or had failing tests.
    • links to details about what code or flag changesets are in each snapshot of code.
    • links to details about which tests failed and why.
    • data on which tests are problematic.
